# 世界観・ビジュアル拡張のアイデア（将来案）

Quest4Bugs を「3つのゲーム集」から「ひとつの昆虫探検の世界」へ広げるための検討メモ。
現時点では未実装の構想であり、優先度・実装可否を判断するためのたたき台。

## 1. なぜ世界観を広げるか

- 現状はポータル（一覧）から各ゲームへ飛ぶ構造で、機能的には十分だが「探検している感」「ひとつの世界に住んでいる感」が薄い。
- 子どもの継続動機は「次の場所に行きたい」「この地域の虫を集めたい」という空間的な好奇心と相性が良い。
- すでに昆虫図鑑が3ゲームに**分類で割り振られている**（蝶・蛾＝かんじ／甲虫＝けいさん／その他＝えいたんご）。この分担は、そのまま「地域ごとに住む虫が違う」という世界地図の土台になる。

## 2. 中心アイデア：ワールドマップ

ポータルを「俯瞰マップ」にし、地域（エリア）を歩いて学習コンテンツを選ぶ。

```
        〜 むしの しま 〜
   ┌──────────────────────────┐
   │   🦋 チョウの森           │  ← かんじ（鱗翅目：蝶・蛾）
   │      （ひらがな〜6年）     │
   │                          │
   │   🪲 カブトの山      🐝 川と草はら │  ← けいさん（甲虫） / えいたんご（その他）
   │   （筆算・九九）      （英検 各級＝フィールド）  │
   │                          │
   │   🏆 チャレンジの塔        │  ← 認定テスト・受験・ヌシ戦
   └──────────────────────────┘
```

- **3つの主要エリア = 3ゲーム**。各エリアのテーマ生物が、その地域で集まる昆虫と一致しているので説明が要らない。
  - チョウの森：かんじ。アゲハ・シジミ・タテハ・ゼフィルス・ガが舞う森。学年が「森の奥」へ進むイメージ。
  - カブトの山：けいさん。クワガタ・カブト・カミキリの山。筆算レベル・九九段が「登る高さ」。
  - 川と草はら：えいたんご。トンボ・バッタ・ハチ・水生昆虫。英検の級＝フィールド（にわ→いけ→はやし→やま→どうくつ→ジャングル→アマゾン）が川沿いに並ぶ。
- **進捗がマップに反映**：マスターした学年/級/段に応じて、その地域が「ひらけていく」（霧が晴れる・色がつく・虫が増える）。ロック中の奥地は暗く、「🏆チャレンジで ひらこう」と誘導（現在の範囲ロック可視化と同じ思想）。
- **採集密度の可視化**：地域ごとに「この森の虫 53/278しゅるい」のようにコレクション進捗を表示。集めるほど地域が賑やかになる。

## 3. 既存資産との接続（実装が現実的な理由）

- ポータルは既に「ハブ」。マップはその見た目を差し替えるUI層で、データ構造の変更は不要。
- 進捗要約（`progressSummary`）・採集数・連続日数・称号はダッシュボードで集計済み → マップ上のラベルにそのまま流用できる。
- 昆虫の地域割り当ては `shared/reward.js` の `gameFor()`（order基準）で確定済み → 地域＝ゲーム＝虫の対応が既にある。
- 夜表示（えいたんごの「🌙よるは よるの虫が出やすい」）が既にあり、**時間帯で世界が変わる**仕掛けの素地がある。

## 4. 段階的な実装案（小さく始める）

1. **第1段**：ポータルを「マップ風カード」に。3エリアを横並びのイラストカードにし、各カードに progressSummary・採集数・🏅称号を載せる（≒ 今のダッシュボードを地図っぽく見せるだけ）。データ変更ゼロ。
2. **第2段**：SVG/イラストの俯瞰マップに置き換え、エリアをタップで該当ゲームへ。ロック/解放・霧晴れ演出を進捗連動で。
3. **第3段**：「キャラ（あいぼうの虫）がマップ上を歩く」演出、季節・天気・昼夜でマップの見た目が変わる、地域限定イベント（「今日は森にレアが出やすい」＝抜き打ち復習をマップ上に表現）。

## 5. その他のビジュアル/世界観アイデア

- **標本箱メタファ**：図鑑を「標本箱が埋まっていく」見せ方に。地域ごとの標本箱。
- **季節**：実日付で季節を変え、季節限定の虫の出現（春＝ギフチョウ、夏＝カブト、秋＝赤とんぼ…）。学習の「旬」を演出。
- **あいぼうの成長を世界に出す**：かんじの相棒（🥚→🐛→🪲）をマップ上の連れ歩きキャラに拡張し、全ゲーム共通の相棒に。
- **描画アーキタイプの拡充**：現在は科ごとのアーキタイプ＋色替え。蝶のアゲハ/シジミ/セセリ等は分化済みだが、甲虫やトンボの形をさらに増やすと地域の見た目の多様性が上がる。
- **音**：地域BGM・虫の声（鈴虫・蝉）で没入感。

## 6. 留意点

- 子ども（5歳コース含む）には情報過多にならないよう、マップは「大きな3エリア＋現在地」程度のシンプルさを保つ。
- 世界観演出はあくまで学習導線の補助。コア（問題→正解→採集→進捗）の手触りを邪魔しないこと。
- 既存セーブとの後方互換を維持（マップ化はUI層のみで、保存データ構造は変えない方針）。
